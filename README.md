## 알고리즘(algorithm)

|제목|설명|
|-----|--------------|
|adjacent_find|지정한 조건을 충족하는 인접 요소 두 개를 검색하는 알고리즘의 한 함수|
|all_of|지정한 범위에서 각 요소에 조건에 해당하는 것이 있는지 확인하는 함수|
|any_of|요소의 지정된 범위에 조건이 있는지 확인하는 함수|
